# Introduction to Generaative AI (GenAI)

- History of GenAI
  - 1956
    - Artificial Intelligence
  - 1997
    - Machine Learning
      - Subset of AI
      - Machines can learn from existing data
        - Improve data or make predictions from data
  - 2012
    - Deep Learning
      - Neural networks are used to process data and make decisions
  - 2021 
    - Generative AI
      - Create new content based on prompts or existing data
  
  - LLMs
    - Built using a neural network architecture
      - Takes in inputs (input layer)
      - Several hidden layers that break down inputs
      - Produces output (output layer)
  
  - Foundation Model
    - A large-scale AI system trained on a massive, diverse dataset that can be adapted to a wide variety of downstream tasks
      - ex. GPT-4, GPT-5, Sonnet 4.6, Opus 4.8, etc.
